📚 帮助指南

蔚来智能销售助手使用指南

欢迎使用蔚来智能销售助手，这是一个基于AI的销售管理系统，能够帮助销售人员更高效地管理客户线索、生成个性化话术，并提供智能分析和建议。

🚀 快速开始

1. 品牌配置

首次使用时，点击右上角 ⚙ 配置 按钮，设置您的品牌信息和AI模型参数。配置完成后，所有界面文案和AI生成的话术将自动适配您的品牌。

2. 数据总览

在 📊 数据总览 页面，您可以查看：

  • 关键绩效指标（KPI）
  • 销售漏斗分析
  • 竞品威胁指数
  • AI智能提醒
  • 客户情绪分布

3. 客户管理

切换到 👥 客户管理 页面进行客户操作：

  • 从左侧列表选择客户
  • 查看客户详细信息和时间线
  • 使用右侧AI建议生成话术
  • 管理客户标签和状态

🤖 AI功能说明

AI话术生成

系统支持多种话术生成场景：

场景 说明 适用阶段
初次接触 生成友好的开场白，建立信任 初步联系
产品介绍 根据客户偏好定制产品亮点 线索跟进
竞品对比 突出蔚来优势，化解疑虑 邀约到店
价格谈判 提供灵活方案，促成成交 价格谈判

智能分析

AI会自动分析：

  • 客户情绪状态
  • 购买意向强度
  • 竞品关注度
  • 最佳跟进时机
  • 战败原因预测

⚙ 配置选项

品牌设置

  • 品牌名称：如蔚来、小鹏、理想等
  • 门店位置：具体门店地址
  • 核心卖点：品牌独特优势

AI模型配置

支持多种AI模型：

  • DeepSeek Chat：推荐，国内稳定
  • 通义千问：阿里云模型
  • 智谱GLM：清华系模型
  • 月之暗面：Kimi模型
  • 演示模式：无需API Key

📊 数据面板说明

KPI指标

  • 线索总数：本月新增客户线索
  • 邀约到店率：线索转化为到店的比例
  • 成交转化率：最终成交比例
  • AI话术采纳率：使用AI生成话术的比例

销售漏斗

展示从线索获取到成交的完整流程转化率，帮助识别销售瓶颈。

竞品威胁指数

基于客户关注度和市场数据，量化各竞品的影响力。

🎯 最佳实践

日常使用建议

  1. 每天查看AI洞察，优先处理高意向线索
  2. 定期检查情绪负面的客户，及时调整沟通策略
  3. 利用AI生成的话术，但结合个人经验适当调整
  4. 及时更新客户状态，确保数据准确性

效率提升技巧

  • 使用批量战败功能清理无效线索
  • 设置客户标签，便于分类管理
  • 关注时间线记录，了解客户全貌
  • 定期查看操作日志，优化销售流程

🔧 故障排除

常见问题

  • AI生成失败：检查API Key配置和网络连接
  • 数据不更新：刷新页面或检查浏览器缓存
  • 界面显示异常：尝试清除浏览器缓存

技术支持

如遇到技术问题，请联系技术支持团队或查看系统日志获取更多信息。

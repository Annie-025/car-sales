# AI 智能销售助手
## 产品使用说明书 · v3.0

---

## 一、产品概述

AI 智能销售助手是一款面向新能源汽车品牌门店的**半自主销售辅助系统**，聚焦线索跟进与邀约阶段，通过 AI 自动执行确定性任务、辅助销售进行高价值决策，人工保留最终确认权。

**设计原则**：AI 建议 → 人工确认 → 系统执行。AI 绝不绕过销售顾问直接对外发送信息。

---

## 二、快速上手（3步开始使用）

### 第1步：配置品牌与AI模型

点击右上角 **⚙ 配置** 按钮，填写：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 品牌名称 | 您的汽车品牌 | 蔚来、小鹏、理想、问界 |
| 门店位置 | 门店名称 | 北京朝阳店 |
| 核心卖点 | 品牌差异化优势（AI生成话术时引用） | 换电网络、NIO Life权益 |
| AI 模型 | 选择大模型供应商 | DeepSeek（推荐） |
| API Key | 对应模型的密钥 | 留空则使用演示模式 |

**推荐模型：DeepSeek Chat**
- 国内服务器直连，无需代理
- 免费额度：每天50万 Token，足够演示使用
- 申请地址：platform.deepseek.com

### 第2步：查看数据总览

进入**📊 数据总览**页面，系统自动展示：
- 本月关键指标（线索数、到店率、转化率、话术采纳率）
- 销售漏斗各阶段流转数据
- AI 今日待处理提醒（按优先级排序）
- 低质线索批量战败建议（若有）

### 第3步：开始客户跟进

切换至**👥 客户管理**，点击左侧线索进入工作台：
1. 查看客户360°画像（标签、情绪、竞品风险）
2. 点击「🤖 生成话术」获取3种风格的个性化开场白
3. 选择满意的话术后点击「📋 复制话术」一键复制
4. 点击「📅 安排跟进」同步至日历提醒

## 三、核心功能详解

### 3.1 数据总览仪表盘

**今日 AI 洞察条**：基于当日数据，AI 自动生成一句话核心建议，含可点击的快捷处理按钮。

**低质线索批量战败**（半自主特性）：
- 系统自动识别评分 ≤40 且超30天无响应的线索
- 橙红色横幅提示数量，展示客户姓名与评分
- 点击「一键批量战败」打开归因弹窗：
  - AI 为每条线索分析最可能的战败原因并预选
  - 销售可逐条调整，也可使用 AI 推荐直接确认
  - 点击「确认批量战败」后系统统一归档，操作记录在日志中可查

> **注意**：批量战败为不可逆操作（演示版），正式系统中可在操作日志中恢复。

**销售漏斗**：可视化5阶段转化数据，点击各阶段数字可筛选对应客户。

**竞品威胁指数**：统计当月各竞品被提及次数，数值越大表示当月竞争压力越大。

### 3.2 客户工作台

#### 客户画像区

| 信息类型 | 内容 | 可视化方式 |
|----------|------|------------|
| 身份标签 | 购车类型、年龄性别、职业场景 | 蓝色标签 |
| 情绪标签 | 积极 / 中性 / 负面 | 颜色+emoji标签 |
| 竞品标签 | 正在对比的竞品 | 红色标签 |
| 预算标签 | 意向预算范围 | 橙色标签 |
| AI评分 | 0-100 意向强度综合评分 | 彩色徽章 |

**阶段进度条**：5个销售阶段以绿色（已完成）/ 蓝色闪烁（当前）/ 灰色（未到）视觉化展示。

**快捷数据栏**：一行显示意向评分、沟通次数、当前情绪、竞品威胁值、上次联系时间。

#### 沟通时间线

每条沟通记录包含：
- 📞 电话 / 💬 企微 / 🏪 到店 类型图标
- 情绪星级评分（1-5星）
- AI提取的关键词标签（如「价格质疑」「换电感兴趣」）

#### 主操作按钮

| 按钮 | 触发动作 | AI参与程度 |
|------|----------|------------|
| 🤖 生成话术 | 打开右侧AI面板，调用AI生成3种话术 | AI生成，人工选择 |
| 📅 安排跟进 | 模拟同步跟进计划至日历 | AI自动排程 |
| 📩 发邀请 | 模拟发送试驾邀请短信 | AI起草，人工确认后执行 |
| ❌ 标记战败 | 打开战败归因弹窗 | AI推荐原因，人工选择 |

#### AI 建议栏（底部）

页面底部常驻 AI 建议条，根据当前客户的阶段和情绪，实时展示最优下一步行动建议。点击「刷新」重新计算。

### 3.3 右侧 AI 面板（4个标签页）

**🤖 AI 建议**
- 意向评分环形图（含4维度雷达）
- AI话术推荐区（3种风格，可切换，选中后显示复制按钮）
- 下一步行动计划（时间+任务+优先级）

**🏁 竞品分析**
- 竞品威胁度横向条形图
- AI生成针对当前竞品的话术要点（3条，每条≤25字）
- 品牌核心差异点（自动读取品牌配置的核心卖点）

**📅 计划**
- 本周跟进计划时间线
- AI自动任务状态（已完成 / 可一键执行）

**📋 日志**
- 全部 AI 操作 + 人工操作记录
- 区分 🤖 AI 操作（蓝色左边线）和 👤 人工操作（绿色左边线）

### 3.4 话术生成——进度反馈机制

点击「生成话术」后，系统展示实时进度，避免用户误以为卡死：

```
⟳ 分析客户画像与历史沟通
○ 匹配竞品话术策略
○ 生成个性化话术变体
⏱ 已用时 2s · 正常响应中...
```

- 0-3秒：绿色"正常响应中"
- 3-8秒：橙色"模型思考中，请稍候..."
- 8-15秒：红色"响应较慢，网络可能不稳定"
- >15秒：显示"建议切换演示模式"快捷按钮

若 API 调用失败，界面显示具体错误信息 + 「重试」+ 「切换演示模式」两个按钮。

## 六、常见问题

**Q：演示模式和真实模式有什么区别？**
A：演示模式使用内置预设数据，响应约1秒，无需网络。真实模式调用真实AI API，响应时间取决于网络和模型，通常2-8秒。两种模式的界面和交互完全一致。

**Q：生成话术时一直显示"正在分析"怎么办？**
A：查看计时器颜色。若超过8秒仍无响应，点击「切换演示模式」可立即使用预设话术继续演示。常见原因：API Key 无效、网络限制、账户额度用完。

**Q：批量战败后能恢复吗？**
A：演示版本中不可恢复（刷新页面重置所有数据）。正式系统中操作日志支持逐条恢复。

**Q：如何切换到其他品牌进行演示？**
A：点击右上角「⚙ 配置」→ 修改品牌名称和核心卖点 → 保存。所有文案和AI提示词即时更新，无需刷新页面。
